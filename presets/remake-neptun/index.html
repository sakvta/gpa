Neptun Remake

This is a free After Effects project file (.aep) built for learning. Study composition structure (precomps), layer organization, effect stacks, and timing/easing in a real project.

How to use

  1. Unzip the downloaded file.
  2. Open the .aep project in After Effects.
  3. If you see missing footage/assets, use Replace Footage… in the Project panel.
  4. Import or copy the compositions you need into your own project and customize them.
